# Project Constitution

## Purpose
NammaRoute is an AI-powered smart public transport planner for Hyderabad commuters combining HMRL Metro and TSRTC Bus services.

## Core Principles
- User-first design
- Real official fare accuracy
- Eco-conscious routing
- Multi-modal transit integration

## Scope
City-level solution for Hyderabad, India.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- AI: Claude API
- Maps: Leaflet.js + OpenStreetMap
- CI/CD: GitLab CI, Pre-commit Hooks
