# User Manual

## Hyderabad Smart Public Transport Planner

### Overview

The Hyderabad Smart Public Transport Planner helps commuters find the best public transport routes based on travel time, cost, and crowd levels.

---

## Features

* Route Search
* Fare Estimation
* AI-Powered Route Recommendations
* Crowd Estimation
* Alternative Route Suggestions
* Interactive Map View

---

## How to Use

### Step 1: Open the Application

Launch the application in your web browser.

### Step 2: Enter Journey Details

* Enter the Source location.
* Enter the Destination location.

### Step 3: Search Routes

Click the **Search Route** button.

### Step 4: View Recommendations

The system displays:

* Fastest Route
* Cheapest Route
* Least Crowded Route

### Step 5: AI Travel Assistant

Users can ask travel-related questions such as:

* "What's the cheapest way to reach HITEC City?"
* "Suggest the least crowded route to Secunderabad."

### Step 6: Check Fare Details

View estimated travel costs for each route.

### Step 7: Select Route

Choose the route that best matches your preference and begin your journey.

---

## System Requirements

* Modern Web Browser (Chrome, Edge, Firefox)
* Internet Connection

---

## Troubleshooting

### Route Not Found

Verify the source and destination names.

### AI Assistant Not Responding

Check internet connectivity and API configuration.

### Map Not Loading

Refresh the page and ensure map services are accessible.
